# Acceptance Checklist

- Homepage answers "What is happening near me that I would actually care about?"
- Every event card shows source attribution and an original source link.
- Mock provider works without external API keys.
- Filters are URL-driven and deterministic.
- Ranking, classification, normalization, and dedupe are covered by tests.
- Saved events persist locally in the browser.
- The sources page explains what is enabled and what is planned next.
