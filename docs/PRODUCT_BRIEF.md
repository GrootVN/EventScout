# Event Scout Product Brief

## Problem

People who are new to a city do not know where to find local events. Events are scattered across event platforms, social media, venue websites, city calendars, university pages, newsletters, and word of mouth.

Existing event apps often show generic listings, but they do not solve the newcomer problem:

- What is happening near me?
- Which events match my interests?
- Which events are good for going alone?
- Which are free or cheap?
- Which are good for meeting people?
- Where did this event come from?
- Is the event still trustworthy?

## Product

Event Scout is a unified event discovery app that collects events from many sources and presents them in one searchable, interest-based interface.

## Target users

- New residents
- College students
- Interns
- Remote workers
- Travelers
- People trying to make friends
- People looking for things to do this weekend

## Core promise

"Find what is happening around you, based on your interests, with the original source always visible."

## MVP features

- Browse events by city
- Filter by interest
- Filter by date
- Filter by price
- View event details
- Open original source
- Save events
- See source badges
- Use mock data with no external API keys
- Support pluggable event source providers

## Non-goals for MVP

- Ticket purchases
- Payments
- Social network features
- Complex user profiles
- Fully automated scraping from all websites
- Unsupported scraping of social platforms
